    HTML (HYPER TEXT MARKUP LANGUAGE): It is used as a way to display information on webpages in your web browser and it is markup language which A markup language is a set of rules that defines how the layout and presentation of text and images should appear in a digital document. It allows structuring documents, adding formatting, and specifying how different elements should be displayed (or “rendered”) on webpages.

    This structuring helps search engines like Google understand the information on websites Some other markup language examples include: XML SGML XHTML Programming languages are used to create functional and dynamic web applications. Markup languages focus primarily on the presentation and structure of content. They are static and don’t use logic or algorithms.

    Semantic Markup

    Semantic markup, also known as semantic HTML, defines sections of a webpage to help browsers, search engines, and developers better understand the content of that webpage.
